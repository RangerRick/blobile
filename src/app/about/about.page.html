About Blases Loaded

Blases Loaded is a little unofficial mobile app for watching Blaseball games, written by Benjamin Reed.

Acknowledgements

Blases Loaded wouldn't be possible without a bunch of folks:

  • The Game Band, for making Blaseball
  • Ionic and Capacitor, the mobile tools used to make this app
  • SVG baseball diamond and peanut from Wikimedia
  • API help, testing, and proxy from SIBR and especially ch00beh for helping test -- check out their live Hawaii Fridays Twitch broadcasts during the season
